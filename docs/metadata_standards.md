# Metadata Standards

## Common Fields

### Candidate Metadata

* candidate_id (unique ID)
* name
* email

### Job Metadata

* job_id
* role_name

### AI Metadata

* model_version (e.g., ATS_v1)
* timestamp (processing time)

### Evaluation Metadata

* ats_score
* screening_score
* interview_score
* final_decision

---

## Purpose

Metadata helps:

* Track candidate progress
* Maintain audit logs
* Support model improvement
* Enable debugging
