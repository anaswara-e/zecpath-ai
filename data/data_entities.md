# Zecpath Data Entities

This document explains the main data structures used in the AI system.

## Candidate Profile
This represents a job candidate.
It includes:
- Name
- Email
- Skills
- Work experience
- Education

## Job Profile
This represents a job opening.
It includes:
- Job title
- Required skills
- Experience required
- Location
- Job type

## Skill Object
A skill is a simple item like:
Python, React, Sales, Communication

## Experience Object
This represents work history:
- Company name
- Role/designation
- Number of years worked
